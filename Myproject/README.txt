The main feature of this application is to handle the request about the 
maximum length of fibonacci sequence which will later be generated. After
users submit their request to server side, with the request-type "add",
to generate a random number as a job number. Meanwhile, fibonacci sequence
will be generated according to the length by users input. In the end, it
will jump to the third page to show those fibonacci sequence on web page.

Please follow these instructions:
1, enter a number as the length of fibonacci sequence.
2, Press submit to proceed the request until the second page, it will delay
10 seconds to jump to the third page.
